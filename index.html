Selected Publications

Exploring Kolmogorov-Arnold networks for realistic image sharpness assessment

Shaode Yu, Ze Chen, Zhimu Yang, Jiacheng Gu, Bizu Feng, Qiurui Sun*

Accepted by ICASSP 2025

Kolmogorov-Arnold Network Machine Learning Image Quality Assessment

This paper underlines that accurate score prediction is key for evaluating realistic image sharpness using informative features. Recently, Kolmogorov-Arnold Networks (KANs) have excelled in data fitting. In this study, the authors introduce a novel Taylor series-based KAN (TaylorKAN) that uses Taylor series expansion to model complex nonlinear relationships between image features and quality scores. They test various KANs on four databases (BID2011, CID2013, CLIVE, and KonIQ-10k) using 15 mid-level and 2048 high-level features. Results indicate that KANs generally outperform support vector regression (SVR), with TaylorKAN achieving the best performance when mid-level features are used. This work is the first to apply KANs to image quality assessment and offers guidance for selecting and enhancing KANs in related tasks.
